# f250_wamv_visual

This local Gazebo Classic model wraps the WAM-V mesh from OSRF VRX:

- Source: https://github.com/osrf/vrx/tree/gazebo_classic/wamv_description/models/WAM-V-Base
- License: Apache-2.0, inherited from https://github.com/osrf/vrx
- Files retained here:
  - `mesh/WAM-V-Base.dae`
  - `mesh/WamV.png`

The SDF wrapper is local to the F250 maritime UAV simulation and adds simple box collisions for stable Gazebo loading.
